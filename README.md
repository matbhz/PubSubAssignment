# PubSub Assignment

## Endpoints:

* POST "/api/{

* POST /api/:topic/:subscriber
	
* DELETE /api/:topic/:subscriber 
	
* GET /api/:topic/:subscriber

## How to run the source code:
On the root of the project, where `main.go` is located, run:

$ go run main.go

## Required  dependency
### Run:
$ go get github.com/gorilla/mux
